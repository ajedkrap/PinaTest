# PinaTest

This repository is meant to fulfill the test for Pina

## Installation

Use the package manager [Node.Js](https://nodejs.org/en/) to PinaTest.

```bash
git clone https://github.com/ajedkrap/PinaTest.git
yarn install
yarn android
```
